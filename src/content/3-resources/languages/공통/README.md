# 언어학습 공통 템플릿

이 폴더는 다양한 언어를 체계적으로 학습하기 위한 템플릿들을 포함합니다.

## 📋 사용 가능한 템플릿

### 1. 10분*언어학습*템플릿.md

- **용도**: 일일 10분 언어학습용
- **포함 내용**:
  - 새 단어/표현 학습
  - 문법 연습
  - 듣기/읽기 연습
  - 말하기/쓰기 연습
  - 학습 후기 및 복습 계획

### 2. 빠른*복습*템플릿.md

- **용도**: 5분 빠른 복습용
- **포함 내용**:
  - 어제 배운 단어 복습
  - 문법 복습
  - 빠른 연습
  - 내일 학습 계획

### 3. 주간*학습*계획\_템플릿.md

- **용도**: 주간 학습 계획 및 성과 관리
- **포함 내용**:
  - 주간 목표 설정
  - 일별 학습 계획
  - 주간 성과 평가
  - 다음 주 개선사항

## 🚀 사용 방법

1. **일일 학습**: `10분_언어학습_템플릿.md`를 복사하여 해당 언어 폴더에 붙여넣기
2. **빠른 복습**: `빠른_복습_템플릿.md`를 사용하여 이전 학습 내용 복습
3. **주간 계획**: `주간_학습_계획_템플릿.md`로 주간 학습 계획 수립

## 💡 학습 팁

- **일관성**: 매일 같은 시간에 학습하는 것이 중요합니다
- **반복**: 배운 내용을 정기적으로 복습하세요
- **실용성**: 일상에서 사용할 수 있는 표현을 우선적으로 학습하세요
- **즐거움**: 학습을 즐겁게 만드는 방법을 찾아보세요

## 📁 언어별 폴더 구조

각 언어 폴더에는 다음과 같은 구조로 학습 자료를 정리하는 것을 권장합니다:

```
언어명/
├── 일일학습/
│   ├── 2024-01-01_학습노트.md
│   └── 2024-01-02_학습노트.md
├── 복습/
│   └── 복습노트.md
├── 문법/
│   └── 문법정리.md
├── 단어장/
│   └── 단어모음.md
└── 연습/
    └── 연습문제.md
```
