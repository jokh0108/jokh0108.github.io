# AI 언어학습 명령어 가이드

## 📋 사용자가 AI에게 했던 주요 명령들

### 1. 언어 학습 요청 명령어

#### 기본 학습 요청

```
"[언어명] 초보야 학습시켜주고 기록으로 남겨"
```

- **예시**: "독일어 초보야 학습시켜주고 기록으로 남겨"
- **효과**: 해당 언어의 초보자용 수업을 진행하고 학습 기록 파일 생성

#### 다음 수업 요청

```
"다음"
```

- **효과**: 이전 수업에 이어서 다음 단계의 수업 진행

### 2. 템플릿 수정 요청 명령어

#### 발음기호 추가 요청

```
"템플릿 부분에 발음기호 표기도 같이하라고 넣어"
```

- **효과**: 학습 템플릿에 IPA/발음기호 표기 컬럼 추가
- **적용된 템플릿**: 10분*언어학습*템플릿.md, 빠른*복습*템플릿.md

#### Input/Output 구조 추가 요청

```
"10분 언어 학습 템플릿에
윗부분에는 매일 학습한 내용을 채울 수 있도록 하고 (소비, input),
아래는 내가 그 학습내용을 직접 꺼내봄으로써 사용/활용(소화, output)할 수 있는 템플릿으로 만들어"
```

- **효과**: 학습 템플릿을 INPUT(소비)과 OUTPUT(소화) 구조로 분리
- **적용된 템플릿**: 10분*언어학습*템플릿.md

```

### 4. AI 활용 방법 문의

```

## 🎯 효과적인 AI 명령어 패턴

- ❌ "도와줘"
- ✅ "독일어 초보야 학습시켜주고 기록으로 남겨"

### 2. 목적 명시

- ❌ "언어 배우고 싶어"
- ✅ "10분 정도 공부할 수 있는 템플릿 만들어봐"

### 3. 결과물 요청

- ❌ "가르쳐줘"
- ✅ "학습시켜주고 기록으로 남겨"

### 4. 수정 요청

- ❌ "바꿔줘"
- ✅ "템플릿 부분에 발음기호 표기도 같이하라고 넣어"

## 📝 자주 사용할 명령어 템플릿

### 언어 학습 시작

```
"[언어명] [수준] 학습시켜주고 기록으로 남겨"
```

### 다음 단계 진행

```
"다음"
```

### 특정 주제 학습

```
"[언어명] [주제]에 대해 학습시켜주고 기록으로 남겨"
```

### 템플릿 수정

```
"템플릿에 [수정사항] 추가해줘"
```

### 복습 요청

```
"[언어명] 복습시켜주고 기록으로 남겨"
```

### AI 활용 학습

```
"[언어명]으로 [내용] 읽어보라고 시켜줘"
```

## 🔄 반복 사용 가능한 명령어

### 1. 일일 학습

- "오늘 [언어명] 학습시켜주고 기록으로 남겨"
- "다음"

### 2. 주간 계획

- "이번 주 [언어명] 학습 계획 세워줘"
- "주간 템플릿 만들어줘"

### 3. 복습

- "[언어명] 복습시켜주고 기록으로 남겨"
- "빠른 복습 템플릿 사용해서 복습해줘"

### 4. 템플릿 관리

- "템플릿에 [항목] 추가해줘"
- "템플릿 수정해줘"

## 💡 명령어 사용 팁

### 1. 명확한 언어 지정

- 항상 학습하고 싶은 언어를 명시하세요
- 예: "독일어", "일본어", "중국어" 등

### 2. 수준 명시

- 초보, 중급, 고급 등 수준을 알려주세요
- 예: "독일어 초보", "일본어 중급"

### 3. 시간 제한

- 학습 시간을 명시하세요
- 예: "10분", "5분", "30분"

### 4. 결과물 요청

- 항상 "기록으로 남겨" 또는 "파일로 만들어줘" 추가
- 학습 내용을 파일로 저장받을 수 있습니다
