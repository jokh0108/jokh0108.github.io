## 📥 INPUT (소비) - 학습한 내용

### 1. 새로 배운 단어/표현 (3분)

| 단어/표현 | 발음 | IPA/발음기호 | 의미 | 예문 |
| --------- | ---- | ------------ | ---- | ---- |
| こんにちは | 곤니치와 | /koɴnit͡ɕiwa/ | 안녕하세요 | こんにちは！元気ですか？ |
| おはよう | 오하요 | /ohajou/ | 좋은 아침 | おはよう！今日もよろしく！ |
| ありがとう | 아리가토 | /aɾiɡatou/ | 감사합니다 | ありがとうございます！ |
| すみません | 스미마셍 | /sumimasen/ | 죄송합니다/실례합니다 | すみません、お手伝いください |
| さようなら | 사요나라 | /sajonaɾa/ | 안녕히 가세요 | さようなら、また明日！ |

### 2. 문법/구문 연습 (3분)

**학습한 문법**: 히라가나 あ행과 기본 문장 구조

- 설명: 히라가나는 일본어의 기본 표음문자, あ행은 모음 5개
- 예문:
  - あ (아) - 사과 りんご
  - い (이) - 좋다 いい  
  - う (우) - 집 うち
  - え (에) - 그림 え
  - お (오) - 차 お茶
- 발음/발음기호: 위의 IPA 표기 참조
- 내가 만든 문장: こんにちは！私は[이름]です。(안녕하세요! 저는 [이름]입니다)

### 3. 듣기/읽기 연습 (2분)

**연습 내용**: 기본 인사말 듣기

- 이해한 내용: こんにちは는 낮 인사, おはよう는 아침 인사
- 모르는 부분: 元気ですか？(잘 지내세요?) - 다음에 배울 내용
- 새로 배운 발음: 일본어의 'ん' 발음은 비음

### 4. 말하기/쓰기 연습 (2분)

**연습 내용**: 인사말 말하기 연습

- 내가 말한/쓴 내용: こんにちは！ありがとう！すみません！さようなら！
- 발음 연습한 부분: 'ん' 발음, 'つ' 발음
- 개선할 점: 장음(ー) 발음을 더 정확하게 하기

## 📤 OUTPUT (소화) - 학습 내용 활용

### 1. 단어/표현 활용 연습 (2분)

**오늘 배운 단어로 문장 만들기**:

- こんにちは를 사용한 문장: こんにちは！初めまして。(안녕하세요! 처음 뵙겠습니다)
- ありがとう를 사용한 문장: ありがとう！助かりました。(고마워요! 도움이 됐어요)
- すみません를 사용한 문장: すみません、教えてください。(죄송한데, 가르쳐 주세요)

**실제 상황에서 사용해보기**:

- 상황 1: 일본인을 만났을 때 - "こんにちは！おはよう！"
- 상황 2: 도움을 받았을 때 - "ありがとうございます！"
- 상황 3: 헤어질 때 - "さようなら！また明日！"

### 2. 문법 활용 연습 (2분)

**학습한 문법으로 대화 만들기**:

- 질문: こんにちは！お名前は？(안녕하세요! 이름이 뭐예요?)
- 답변: こんにちは！私は[이름]です。あなたは？(안녕하세요! 저는 [이름]입니다. 당신은?)
- 추가 질문: ありがとう！元気ですか？(고마워요! 잘 지내세요?)

**문법 패턴 연습**:

- 패턴 1: こんにちは！+ [자기소개]
- 패턴 2: ありがとう + [감사 표현]
- 패턴 3: すみません + [부탁 표현]

### 3. 실전 연습 (1분)

**AI와 대화 연습**:

- AI에게 말할 내용: "こんにちは！私は初心者です。(안녕하세요! 저는 초보자입니다)"
- 예상 AI 응답: "こんにちは！いいですね！(안녕하세요! 좋네요!)"
- 내가 할 답변: "ありがとう！日本語を勉強しています。(고마워요! 일본어를 공부하고 있어요)"

**자기소개/상황 설명**:

- 오늘의 나: 私は日本語の初心者です。(저는 일본어 초보자입니다)
- 내가 좋아하는 것: 日本語の勉強が好きです。(일본어 공부를 좋아해요)
- 내가 하고 싶은 것: 日本語を上手に話したいです。(일본어를 잘 말하고 싶어요)
