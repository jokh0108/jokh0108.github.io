## 📥 INPUT (소비) - 학습한 내용

### 1. 새로 배운 단어/표현 (3분)

| 단어/표현 | 발음 | IPA/발음기호 | 의미 | 예문 |
| --------- | ---- | ------------ | ---- | ---- |
| 你好 | 니하오 | /niˊ xaʊˊ/ | 안녕하세요 | 你好！你好吗？ |
| 早上好 | 자오상하오 | /tsaʊˊ ʃaŋˋ xaʊˊ/ | 좋은 아침 | 早上好！今天天气不错！ |
| 谢谢 | 시에시에 | /ɕjeˋ ɕjeˋ/ | 감사합니다 | 谢谢你的帮助！ |
| 请 | 칭 | /tɕʰiŋˊ/ | 부탁해요 | 请帮助我！ |
| 再见 | 자이지엔 | /tsaiˋ tɕjenˋ/ | 안녕히 가세요 | 再见！明天见！ |

### 2. 문법/구문 연습 (3분)

**학습한 문법**: 중국어 4성조와 기본 문장 구조

- 설명: 중국어는 성조언어로 4개의 기본 성조가 있음
- 예문:
  - 1성(ˉ): 妈 mā (엄마) - 높고 평평한 음
  - 2성(ˊ): 麻 má (마) - 올라가는 음
  - 3성(ˇ): 马 mǎ (말) - 내려갔다 올라가는 음
  - 4성(ˋ): 骂 mà (욕하다) - 내려가는 음
- 발음/발음기호: 위의 IPA 표기 참조
- 내가 만든 문장: 你好！我是[이름]。(안녕하세요! 저는 [이름]입니다)

### 3. 듣기/읽기 연습 (2분)

**연습 내용**: 기본 인사말 듣기

- 이해한 내용: 你好는 가장 기본적인 인사말, 早上好는 아침 인사
- 모르는 부분: 你好吗？(잘 지내세요?) - 다음에 배울 내용
- 새로 배운 발음: 성조에 따라 단어 의미가 완전히 달라짐

### 4. 말하기/쓰기 연습 (2분)

**연습 내용**: 인사말 말하기 연습

- 내가 말한/쓴 내용: 你好！谢谢！请！再见！
- 발음 연습한 부분: 4가지 성조 발음
- 개선할 점: 3성 발음을 더 정확하게 하기

## 📤 OUTPUT (소화) - 학습 내용 활용

### 1. 단어/표현 활용 연습 (2분)

**오늘 배운 단어로 문장 만들기**:

- 你好를 사용한 문장: 你好！我是新学生。(안녕하세요! 저는 새 학생입니다)
- 谢谢를 사용한 문장: 谢谢你！你真好！(고마워요! 당신 정말 좋아요!)
- 请를 사용한 문장: 请问，你是谁？(실례하는데, 당신은 누구세요?)

**실제 상황에서 사용해보기**:

- 상황 1: 중국인을 만났을 때 - "你好！早上好！"
- 상황 2: 도움을 받았을 때 - "谢谢你！"
- 상황 3: 헤어질 때 - "再见！明天见！"

### 2. 문법 활용 연습 (2분)

**학습한 문법으로 대화 만들기**:

- 질문: 你好！你叫什么名字？(안녕하세요! 이름이 뭐예요?)
- 답변: 你好！我叫[이름]。你呢？(안녕하세요! 저는 [이름]입니다. 당신은?)
- 추가 질문: 谢谢！你好吗？(고마워요! 잘 지내세요?)

**문법 패턴 연습**:

- 패턴 1: 你好！+ [자기소개]
- 패턴 2: 谢谢 + [감사 표현]
- 패턴 3: 请 + [부탁 표현]

### 3. 실전 연습 (1분)

**AI와 대화 연습**:

- AI에게 말할 내용: "你好！我是初学者。(안녕하세요! 저는 초보자입니다)"
- 예상 AI 응답: "你好！很好！(안녕하세요! 좋아요!)"
- 내가 할 답변: "谢谢！我在学中文。(고마워요! 저는 중국어를 배우고 있어요)"

**자기소개/상황 설명**:

- 오늘의 나: 我是中文初学者。(저는 중국어 초보자입니다)
- 내가 좋아하는 것: 我喜欢学中文。(저는 중국어 배우는 것을 좋아해요)
- 내가 하고 싶은 것: 我想说流利的中文。(저는 중국어를 유창하게 말하고 싶어요)
