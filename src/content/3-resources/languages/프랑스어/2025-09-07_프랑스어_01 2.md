## 📥 INPUT (소비) - 학습한 내용

### 1. 새로 배운 단어/표현 (3분)

| 단어/표현 | 발음 | IPA/발음기호 | 의미 | 예문 |
| --------- | ---- | ------------ | ---- | ---- |
| Bonjour | 봉쥬르 | /bon.ˈʒuʁ/ | 안녕하세요 | Bonjour! Comment allez-vous? |
| Bonsoir | 봉스와르 | /bon.ˈswaʁ/ | 안녕하세요 (저녁) | Bonsoir! Bonne soirée! |
| Merci | 메르시 | /mer.ˈsi/ | 감사합니다 | Merci beaucoup! |
| S'il vous plaît | 실부플레 | /sil vu ˈplɛ/ | 부탁해요 | S'il vous plaît, aidez-moi. |
| Au revoir | 오 르브와르 | /o rə.ˈvwaʁ/ | 안녕히 가세요 | Au revoir! À bientôt! |

### 2. 문법/구문 연습 (3분)

**학습한 문법**: 프랑스어 알파벳과 악센트 기호

- 설명: 프랑스어는 라틴 문자 + 악센트 기호 사용 (é, è, ç, â 등)
- 예문:
  - é: café (카페, 커피)
  - è: père (페르, 아버지)
  - ç: français (프랑세, 프랑스어)
  - â: château (샤토, 성)
- 발음/발음기호: 위의 IPA 표기 참조
- 내가 만든 문장: Bonjour! Je suis [이름]. (안녕하세요! 저는 [이름]입니다)

### 3. 듣기/읽기 연습 (2분)

**연습 내용**: 기본 인사말 듣기

- 이해한 내용: Bonjour는 낮 인사, Bonsoir는 저녁 인사
- 모르는 부분: Comment allez-vous? (어떻게 지내세요?) - 다음에 배울 내용
- 새로 배운 발음: 프랑스어의 'r' 발음은 목구멍 굴림음

### 4. 말하기/쓰기 연습 (2분)

**연습 내용**: 인사말 말하기 연습

- 내가 말한/쓴 내용: Bonjour! Merci! S'il vous plaît! Au revoir!
- 발음 연습한 부분: 'r' 발음, 비음('n' 소리)
- 개선할 점: 프랑스어 'r' 발음을 더 정확하게 하기

## 📤 OUTPUT (소화) - 학습 내용 활용

### 1. 단어/표현 활용 연습 (2분)

**오늘 배운 단어로 문장 만들기**:

- Bonjour를 사용한 문장: Bonjour! Je suis nouveau ici. (안녕하세요! 저는 여기 처음입니다)
- Merci를 사용한 문장: Merci! Vous êtes très gentil. (고마워요! 당신 정말 친절해요)
- S'il vous plaît를 사용한 문장: S'il vous plaît, pouvez-vous m'aider? (부탁드려요, 도와주실 수 있나요?)

**실제 상황에서 사용해보기**:

- 상황 1: 프랑스인을 만났을 때 - "Bonjour! Bonsoir!"
- 상황 2: 도움을 받았을 때 - "Merci beaucoup!"
- 상황 3: 헤어질 때 - "Au revoir! À bientôt!"

### 2. 문법 활용 연습 (2분)

**학습한 문법으로 대화 만들기**:

- 질문: Bonjour! Comment vous appelez-vous? (안녕하세요! 이름이 뭐예요?)
- 답변: Bonjour! Je m'appelle [이름]. Et vous? (안녕하세요! 제 이름은 [이름]입니다. 당신은?)
- 추가 질문: Merci! Comment allez-vous? (고마워요! 어떻게 지내세요?)

**문법 패턴 연습**:

- 패턴 1: Bonjour! + [자기소개]
- 패턴 2: Merci + [감사 표현]
- 패턴 3: S'il vous plaît + [부탁 표현]

### 3. 실전 연습 (1분)

**AI와 대화 연습**:

- AI에게 말할 내용: "Bonjour! Je suis débutant. (안녕하세요! 저는 초보자입니다)"
- 예상 AI 응답: "Bonjour! C'est parfait! (안녕하세요! 완벽해요!)"
- 내가 할 답변: "Merci! J'apprends le français. (고마워요! 저는 프랑스어를 배우고 있어요)"

**자기소개/상황 설명**:

- 오늘의 나: Je suis débutant en français. (저는 프랑스어 초보자입니다)
- 내가 좋아하는 것: J'aime apprendre le français. (저는 프랑스어 배우는 것을 좋아해요)
- 내가 하고 싶은 것: Je veux parler français couramment. (저는 프랑스어를 유창하게 말하고 싶어요)
