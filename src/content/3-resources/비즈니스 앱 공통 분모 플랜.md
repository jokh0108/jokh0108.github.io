# 비즈니스 앱 공통 분모 플랜

## 🔐 인증/권한
- **소셜 로그인**: Google, GitHub, Apple 
- **이메일/폰 인증**: OTP, 매직링크
- **RBAC**: 역할 기반 권한 관리
- **JWT + Refresh Token**

## 👥 사용자 관리  
- **프로필**: 아바타, 개인정보, 설정
- **팀/조직**: 멀티테넌트 지원
- **사용자 초대/관리**

## 💳 결제/구독
- **Stripe 통합**: 구독, 일회성 결제
- **플랜 관리**: Free/Pro/Enterprise
- **사용량 제한**: API 호출, 스토리지 등
- **인보이스/영수증**

## 🔔 알림 시스템
- **푸시 알림**: 웹, 모바일
- **이메일 알림**: 트랜잭션, 마케팅
- **인앱 알림**: 실시간 업데이트

## 📁 파일 관리
- **업로드**: 드래그앤드롭, 진행률
- **스토리지**: S3/CloudFlare R2 
- **이미지 최적화**: 자동 리사이징/압축
- **CDN 연동**

## 📧 커뮤니케이션
- **이메일**: SendGrid/Resend
- **SMS**: Twilio
- **템플릿 관리**: 동적 컨텐츠

## 📊 분석/추적
- **사용자 행동**: Mixpanel/PostHog
- **오류 추적**: Sentry  
- **성능 모니터링**: Vercel Analytics
- **A/B 테스팅**

## 🛡️ 보안/규정준수
- **레이트 리미팅**: Redis 기반
- **GDPR**: 데이터 삭제, 내보내기
- **감사 로그**: 모든 액션 추적
- **2FA**: TOTP 지원

## ⚙️ 개발자 경험
- **API 문서**: OpenAPI/Swagger
- **Webhook**: 이벤트 알림
- **CLI 도구**: 배포, 마이그레이션
- **개발 환경**: Docker, 핫 리로드

## 🎨 UI/UX 공통 요소
- **디자인 시스템**: Tailwind + 컴포넌트
- **다크 모드**: 자동 전환
- **반응형**: 모바일 퍼스트
- **접근성**: WCAG 2.1 준수

## 🌍 국제화
- **다국어**: i18n 지원
- **시간대**: 자동 감지/변환
- **통화**: 지역별 표시

## 🚀 마케팅/성장
- **랜딩 페이지**: A/B 테스트 가능
- **SEO**: 메타태그, 구조화 데이터  
- **소셜 공유**: OG 이미지 자동 생성
- **추천 시스템**: 레퍼럴 코드

## 권장 스택
**Next.js + Supabase + Stripe** 스택으로 구현하면 대부분의 SaaS를 1-2주 안에 MVP 출시 가능합니다.

각 모듈별 구체적인 구현이 필요하면 말씀하세요!
