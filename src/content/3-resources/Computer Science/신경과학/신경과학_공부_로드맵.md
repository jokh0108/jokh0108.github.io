# 신경과학 공부 로드맵

## 개요
신경과학은 뇌와 신경계의 구조, 기능, 발달을 연구하는 학문입니다. 이 로드맵은 기초 신경해부학부터 최신 신경과학 연구까지 체계적으로 학습할 수 있도록 구성되었습니다.

## 1단계: 기초 생물학 및 화학 (4-6주)

### 1.1 세포생물학 기초
- [ ] 세포막과 막 전위
- [ ] 이온 채널과 이온 펌프
- [ ] 세포 소기관과 기능
- [ ] 단백질 합성과 분해

### 1.2 생화학 기초
- [ ] 신경전달물질의 화학적 성질
- [ ] 효소와 대사 과정
- [ ] 신호 전달 분자들
- [ ] 막 수송 메커니즘

### 1.3 유전학 기초
- [ ] DNA, RNA, 단백질의 관계
- [ ] 유전자 발현 조절
- [ ] 후성유전학 기초
- [ ] 신경계 발달과 유전자

## 2단계: 신경해부학 (6-8주)

### 2.1 중추신경계 해부학
- [ ] 뇌의 전체 구조와 분할
- [ ] 대뇌피질의 구조와 기능
- [ ] 뇌간과 소뇌
- [ ] 척수의 구조

### 2.2 말초신경계
- [ ] 뇌신경(cranial nerves)
- [ ] 척수신경(spinal nerves)
- [ ] 자율신경계 (교감/부교감)
- [ ] 감각 기관과 신경 연결

### 2.3 신경계 발달
- [ ] 신경관 형성
- [ ] 신경세포 분화와 이주
- [ ] 축삭 성장과 시냅스 형성
- [ ] 신경계 성숙과 가소성

## 3단계: 세포 및 분자 신경과학 (8-10주)

### 3.1 뉴런의 구조와 기능
- [ ] 뉴런의 형태학적 분류
- [ ] 세포체, 수상돌기, 축삭의 특성
- [ ] 글리아 세포의 종류와 기능
- [ ] 혈뇌장벽

### 3.2 신경 신호 전달
- [ ] 안정 전위와 활동 전위
- [ ] 나트륨-칼륨 펌프
- [ ] 전압 개폐 이온 채널
- [ ] 신호 전파와 수초

### 3.3 시냅스 전달
- [ ] 화학적 시냅스 vs 전기적 시냅스
- [ ] 신경전달물질 분비와 수용체
- [ ] 시냅스 후 전위 (EPSP, IPSP)
- [ ] 시냅스 가소성 (LTP, LTD)

### 3.4 주요 신경전달물질 시스템
- [ ] 아세틸콜린 시스템
- [ ] 도파민 시스템
- [ ] 세로토닌 시스템
- [ ] GABA와 글루타메이트
- [ ] 아드레날린/노르아드레날린

## 4단계: 시스템 신경과학 (10-12주)

### 4.1 감각 시스템
- [ ] 체성감각계 (촉각, 압각, 온도감각)
- [ ] 시각계 (망막, 시각피질, 시각 처리)
- [ ] 청각계 (달팽이관, 청각피질)
- [ ] 후각계와 미각계
- [ ] 통증 시스템

### 4.2 운동 시스템
- [ ] 척수 운동회로
- [ ] 운동피질과 피라미드로
- [ ] 기저핵과 운동 조절
- [ ] 소뇌와 운동 학습
- [ ] 반사와 자동 운동

### 4.3 인지 및 고차 기능
- [ ] 주의와 각성
- [ ] 기억 시스템 (작업기억, 장기기억)
- [ ] 학습과 조건화
- [ ] 언어 처리 (브로카, 베르니케 영역)
- [ ] 실행 기능과 전전두엽

### 4.4 정서와 동기
- [ ] 변연계 구조와 기능
- [ ] 편도체와 공포/불안
- [ ] 해마와 기억 형성
- [ ] 보상 시스템과 중독
- [ ] 스트레스 반응과 HPA축

## 5단계: 신경과학 연구 방법론 (6-8주)

### 5.1 전기생리학적 기법
- [ ] 패치클램프 기법
- [ ] 세포외 기록
- [ ] EEG와 MEG
- [ ] 자극 기법 (DBS, TMS, 광유전학)

### 5.2 영상 기법
- [ ] 구조적 MRI
- [ ] 기능적 MRI (fMRI)
- [ ] PET와 SPECT
- [ ] 광학 영상법
- [ ] 2광자 현미경

### 5.3 분자생물학적 기법
- [ ] 면역조직화학
- [ ] In situ hybridization
- [ ] 유전자 조작 기법
- [ ] 바이러스 벡터 활용
- [ ] 단일세포 RNA 시퀀싱

### 5.4 행동 실험 및 분석
- [ ] 동물 행동 실험 설계
- [ ] 인지 과제 설계
- [ ] 신호 검출 이론
- [ ] 통계 분석 방법

## 6단계: 임상 신경과학 (4-6주)

### 6.1 신경계 질환
- [ ] 알츠하이머병과 치매
- [ ] 파킨슨병과 운동 장애
- [ ] 뇌졸중과 뇌손상
- [ ] 간질과 발작
- [ ] 다발성 경화증

### 6.2 정신 장애의 신경과학

- [ ] 조현병의 뇌 변화
- [ ] 자폐 스펙트럼 장애
- [ ] ADHD의 신경 메커니즘
- [ ] 불안 장애의 뇌 회로

### 6.3 신경약물학
