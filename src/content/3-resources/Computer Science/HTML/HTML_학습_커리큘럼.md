## 🚀 즉시 활용 가능한 HTML 패턴

### 완벽한 HTML5 문서 템플릿
```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>페이지 제목</title>
</head>
<body>
    <header>
        <nav><!-- 네비게이션 --></nav>
    </header>
    <main>
        <section><!-- 콘텐츠 --></section>
    </main>
    <footer><!-- 푸터 --></footer>
</body>
</html>
```

### 시맨틱 요소 빠른 참조
- [ ] `<header>` - 페이지/섹션 헤더
- [ ] `<nav>` - 네비게이션 메뉴
- [ ] `<main>` - 주요 콘텐츠
- [ ] `<section>` - 문서의 섹션
- [ ] `<article>` - 독립적인 콘텐츠
- [ ] `<aside>` - 사이드바, 관련 정보
- [ ] `<footer>` - 페이지/섹션 푸터

### 실습 예제 코드
```html
<!-- 완전한 랜딩페이지 예제 -->
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>나의 포트폴리오</title>
</head>
<body>
    <header>
        <h1>홍길동</h1>
        <nav>
            <ul>
                <li><a href="#about">소개</a></li>
                <li><a href="#projects">프로젝트</a></li>
                <li><a href="#contact">연락처</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <section id="about">
            <h2>소개</h2>
            <p>안녕하세요. 웹 개발자 홍길동입니다.</p>
            <img src="profile.jpg" alt="홍길동 프로필 사진">
        </section>
        
        <section id="projects">
            <h2>프로젝트</h2>
            <article>
                <h3>프로젝트 1</h3>
                <p>프로젝트 설명</p>
            </article>
        </section>
        
        <section id="contact">
            <h2>연락처</h2>
            <form>
                <label for="name">이름:</label>
                <input type="text" id="name" required>
                
                <label for="email">이메일:</label>
                <input type="email" id="email" required>
                
                <label for="message">메시지:</label>
                <textarea id="message" required></textarea>
                
                <button type="submit">보내기</button>
            </form>
        </section>
    </main>
    
    <footer>
        <p>&copy; 2025 홍길동. All rights reserved.</p>
    </footer>
</body>
</html>
```
