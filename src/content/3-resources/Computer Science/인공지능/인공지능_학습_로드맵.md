# 인공지능 학습 로드맵

## 개요
인공지능(Artificial Intelligence)은 인간의 지능을 모방하거나 이를 뛰어넘는 기계 지능을 연구하는 분야입니다. 이 로드맵은 AI의 기초 이론부터 최신 딥러닝 기술까지 체계적으로 학습할 수 있도록 구성되었습니다.

### 1.1 선형대수학
- [ ] 벡터와 벡터 연산
- [ ] 행렬과 행렬 연산
- [ ] 고유값과 고유벡터

### 1.2 확률과 통계

- [ ] 조건부 확률과 독립성
- [ ] 중심극한정리와 대수의 법칙
- [ ] 추정과 검정
- [ ] 정보 이론 기초 (엔트로피, KL 발산)
