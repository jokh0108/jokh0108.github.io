## 📋 1일 학습 계획 (8시간)

### 1단계: 신경과학 기초 (2시간)
#### 1.1 신경과학 개요 (30분)
- 신경과학이란?
- 뇌의 진화와 발달
- 신경과학의 연구 방법론
- 현대 신경과학의 주요 발견들

#### 1.2 뇌의 전체적 구조 (45분)
```
중추신경계 (CNS)
├── 뇌 (Brain)
│   ├── 대뇌 (Cerebrum)
│   │   ├── 대뇌피질 (Cerebral Cortex)
│   │   └── 대뇌변연계 (Limbic System)
│   ├── 소뇌 (Cerebellum)
│   └── 뇌간 (Brainstem)
│       ├── 중뇌 (Midbrain)
│       ├── 뇌교 (Pons)
│       └── 연수 (Medulla)
└── 척수 (Spinal Cord)

말초신경계 (PNS)
├── 체성신경계 (Somatic NS)
└── 자율신경계 (Autonomic NS)
    ├── 교감신경계 (Sympathetic)
    └── 부교감신경계 (Parasympathetic)
```

#### 1.3 뇌의 기능적 영역 (45분)
- **전두엽**: 실행 기능, 계획, 의사결정
- **두정엽**: 공간 인식, 감각 통합
- **측두엽**: 청각, 언어, 기억
- **후두엽**: 시각 처리
- **변연계**: 감정, 동기, 기억 형성

### 2단계: 뉴런과 신호 전달 (2시간)
#### 2.1 뉴런의 구조와 기능 (1시간)
```
뉴런의 구조:
┌─────────────────────────────────────┐
│ 세포체 (Cell Body/Soma)              │
│ - 핵 (Nucleus)                     │
│ - 미토콘드리아 (Mitochondria)        │
│ - 소포체 (Endoplasmic Reticulum)    │
└─────────────────────────────────────┘
           │
    수상돌기 (Dendrites)
           │
           ↓
    축삭 (Axon)
           │
           ↓
    축삭말단 (Axon Terminals)
           │
           ↓
    시냅스 (Synapses)
```

#### 2.2 활동전위와 신호전달 (1시간)
- **안정전위**: -70mV
- **탈분극**: Na+ 채널 개방
- **재분극**: K+ 채널 개방
- **불응기**: 신호의 방향성 보장

```
활동전위의 단계:
1. 안정전위 (-70mV)
2. 임계점 도달 (-55mV)
3. 탈분극 (+30mV)
4. 재분극 (다시 -70mV로)
5. 과분극 (-80mV)
6. 안정전위 복원
```

### 3단계: 시냅스와 신경전달물질 (2시간)
#### 3.1 시냅스 전달 메커니즘 (1시간)
```
시냅스 전달 과정:
1. 활동전위가 축삭말단 도달
2. Ca²⁺ 채널 개방
3. 신경전달물질 소포 융합
4. 신경전달물질 방출
5. 수용체 결합
6. 후시냅스 반응
7. 신경전달물질 제거
```

#### 3.2 주요 신경전달물질 (1시간)
- **도파민**: 보상, 동기, 운동 조절
- **세로토닌**: 기분, 수면, 식욕 조절
- **아세틸콜린**: 학습, 기억, 근육 수축
- **GABA**: 주요 억제성 신경전달물질
- **글루타메이트**: 주요 흥분성 신경전달물질
- **노르에피네프린**: 각성, 주의, 스트레스 반응

### 4단계: 학습과 기억의 신경과학 (2시간)
#### 4.1 기억의 종류와 뇌 영역 (1시간)
```
기억 시스템:
├── 감각기억 (0.5-3초)
├── 단기기억/작업기억 (15-30초)
│   └── 해마 (Hippocampus) 관여
└── 장기기억
    ├── 명시적 기억 (선언적)
    │   ├── 일화적 기억 (해마)
    │   └── 의미적 기억 (대뇌피질)
    └── 암묵적 기억 (절차적)
        └── 소뇌, 기저핵
```

#### 4.2 시냅스 가소성과 학습 (1시간)
- **장기강화 (LTP)**: 시냅스 연결 강화
- **장기억제 (LTD)**: 시냅스 연결 약화
- **헤브 학습 규칙**: "함께 발화하는 뉴런은 연결된다"
- **신경 가소성**: 뇌의 구조적/기능적 변화 능력

## 🧠 핵심 개념 실습

### 실습 1: 뇌 영역 매핑
```
과제: 다음 기능들을 담당하는 뇌 영역 찾기
1. 언어 이해 → 베르니케 영역 (측두엽)
2. 언어 생산 → 브로카 영역 (전두엽)
3. 시각 처리 → 후두엽
4. 감정 처리 → 편도체 (변연계)
5. 기억 형성 → 해마 (변연계)
6. 운동 조절 → 소뇌, 기저핵
```

### 실습 2: 뉴런 신호 계산
```
활동전위 전파 속도 계산:
- 유수 축삭: 50-120 m/s
- 무수 축삭: 0.5-2 m/s

예제: 1m 길이의 유수 축삭에서 신호 전달 시간
시간 = 거리 / 속도 = 1m / 100m/s = 0.01초 = 10ms
```

### 실습 3: 학습 효과 분석
```
학습 효과를 높이는 신경과학적 원리:
1. 간격 반복 (Spaced Repetition)
   - 시냅스 강화를 위한 최적 간격

2. 활성 회상 (Active Recall)
   - 기억 회로 강화

3. 다양한 맥락에서 학습
   - 신경 연결의 다양성 증가

4. 충분한 수면
   - 기억 고착화 과정
```
