# 10분 외국어 학습 Claude 프롬프트 모음

## 사용법
각 언어별로 10분 학습 후, 해당 언어 프롬프트를 사용해 Claude와 연습하세요.

## 언어별 프롬프트

### 영어 (English)
```
영어 학습 도우미가 되어주세요. 저는 영어를 학습 중입니다.

1. **단어/표현 연습**: 제가 새로 배운 단어들로 자연스러운 대화를 만들어주세요
2. **문법 연습**: 학습한 문법을 활용한 질문-답변 연습을 도와주세요
3. **실전 대화**: 일상 상황을 가정하고 영어로 대화해주세요
4. **발음 교정**: 제가 영어로 말한 내용의 자연스러운 표현을 제안해주세요

오늘 학습한 내용: [여기에 학습 내용 입력]
```

### 일본어 (日本語)
```
日本語学習のアシスタントになってください。私は日本語を勉強しています。

1. **単語・表現練習**: 今日覚えた単語を使って自然な会話を作ってください
2. **文法練習**: 学習した文法を活用した質問・回答の練習を手伝ってください
3. **実践会話**: 日常的な状況を想定して日本語で会話してください
4. **敬語練習**: 適切な敬語の使い方を教えてください

今日の学習内容: [ここに学習内容を入力]
```

### 중국어 (中文)
```
请做我的中文学习助手。我正在学习中文。

1. **词汇表达练习**: 用我今天学的生词创造自然的对话
2. **语法练习**: 帮我练习今天学的语法，进行问答练习
3. **实际对话**: 假设日常情况，用中文和我对话
4. **声调练习**: 纠正我的发音，建议更自然的表达方式

今天学习的内容: [在这里输入学习内容]
```

### 스페인어 (Español)
```
Por favor, sé mi asistente de aprendizaje de español. Estoy estudiando español.

1. **Práctica de vocabulario**: Crea conversaciones naturales usando las palabras que aprendí hoy
2. **Práctica de gramática**: Ayúdame a practicar la gramática que estudié con preguntas y respuestas
3. **Conversación práctica**: Supongamos situaciones cotidianas y conversemos en español
4. **Corrección de pronunciación**: Sugiere expresiones más naturales para lo que digo en español

Contenido de estudio de hoy: [Introduce el contenido de estudio aquí]
```

### 프랑스어 (Français)
```
Soyez mon assistant d'apprentissage du français. J'apprends le français.

1. **Pratique du vocabulaire**: Créez des conversations naturelles avec les mots que j'ai appris aujourd'hui
2. **Pratique de grammaire**: Aidez-moi à pratiquer la grammaire étudiée avec des questions-réponses
3. **Conversation pratique**: Supposons des situations quotidiennes et conversons en français
4. **Correction de prononciation**: Suggérez des expressions plus naturelles pour ce que je dis en français

Contenu d'étude d'aujourd'hui: [Saisissez le contenu d'étude ici]
```

### 독일어 (Deutsch)
```
Bitte seien Sie mein Deutsch-Lernassistent. Ich lerne Deutsch.

1. **Wortschatz-Übung**: Erstellen Sie natürliche Gespräche mit den Wörtern, die ich heute gelernt habe
2. **Grammatik-Übung**: Helfen Sie mir, die studierte Grammatik mit Fragen und Antworten zu üben
3. **Praktische Unterhaltung**: Nehmen wir alltägliche Situationen an und unterhalten uns auf Deutsch
4. **Aussprache-Korrektur**: Schlagen Sie natürlichere Ausdrücke für das vor, was ich auf Deutsch sage

Heutiger Lerninhalt: [Geben Sie den Lerninhalt hier ein]
```

### 러시아어 (Русский)
```
Пожалуйста, станьте моим помощником в изучении русского языка. Я изучаю русский язык.

1. **Практика словарного запаса**: Создайте естественные разговоры, используя слова, которые я выучил сегодня
2. **Практика грамматики**: Помогите мне практиковать изученную грамматику с вопросами и ответами
3. **Практический разговор**: Предположим повседневные ситуации и поговорим по-русски
4. **Исправление произношения**: Предложите более естественные выражения для того, что я говорю по-русски

Сегодняшнее содержание обучения: [Введите содержание обучения здесь]
```

### 아랍어 (العربية)
```
من فضلك كن مساعد تعلم اللغة العربية. أنا أتعلم اللغة العربية.

1. **ممارسة المفردات**: اصنع محادثات طبيعية باستخدام الكلمات التي تعلمتها اليوم
2. **ممارسة القواعد**: ساعدني في ممارسة القواعد التي درستها بالأسئلة والأجوبة
3. **المحادثة العملية**: لنفترض مواقف يومية ونتحدث بالعربية
4. **تصحيح النطق**: اقترح تعبيرات أكثر طبيعية لما أقوله بالعربية

محتوى الدراسة اليوم: [أدخل محتوى الدراسة هنا]
```

### 태국어 (ภาษาไทย)
```
กرุณาเป็นผู้ช่วยการเรียนภาษาไทยของฉัน ฉันกำลังเรียนภาษาไทย

1. **ฝึกคำศัพท์**: สร้างการสนทนาธรรมชาติโดยใช้คำที่ฉันเรียนรู้วันนี้
2. **ฝึกไวยากรณ์**: ช่วยฉันฝึกไวยากรณ์ที่เรียนด้วยคำถามและคำตอบ
3. **การสนทนาในทางปฏิบัติ**: สมมติสถานการณ์ในชีวิตประจำวันและสนทนาเป็นภาษาไทย
4. **แก้ไขการออกเสียง**: แนะนำการแสดงออกที่เป็นธรรมชาติมากขึ้นสำหรับสิ่งที่ฉันพูดเป็นภาษาไทย

เนื้อหาการเรียนรู้วันนี้: [ใส่เนื้อหาการเรียนรู้ที่นี่]
```

## 심화 학습 프롬프트

### 문법 집중 연습
```
저는 [언어명]의 [특정 문법 포인트]를 연습하고 싶습니다.

1. 이 문법의 기본 규칙을 간단히 설명해주세요
2. 3가지 난이도의 예문을 제공해주세요 (초급/중급/고급)
3. 제가 직접 문장을 만들 수 있도록 상황을 제시해주세요
4. 제 문장을 교정해주세요

학습하고 싶은 문법: [문법 포인트 입력]
```

### 발음 연습
```
[언어명] 발음 연습을 도와주세요.

1. 제가 어려워하는 발음: [발음 기호 또는 단어]
2. 비슷한 한국어 발음으로 설명해주세요
3. 입술과 혀의 위치를 설명해주세요
4. 연습할 수 있는 단어나 문장을 제공해주세요

오늘 연습하고 싶은 발음: [발음 입력]
```

### 실전 상황별 대화
```
[언어명]로 [상황] 상황에서의 대화를 연습하고 싶습니다.

상황 설정:
- 장소: [장소]
- 상대방: [상대방]
- 목적: [대화 목적]

1. 실제와 같은 대화를 시작해주세요
2. 제 답변에 자연스러운 반응을 해주세요
3. 대화 중 더 좋은 표현이 있다면 제안해주세요
4. 문화적 맥락도 함께 설명해주세요

연습하고 싶은 상황: [상황 설명]
```

### 작문 연습
```
[언어명]으로 [주제]에 대한 글을 써보고 싶습니다.

1. 주제에 맞는 유용한 표현들을 미리 알려주세요
2. 글의 구조를 제안해주세요
3. 제가 쓴 글을 교정해주세요
4. 더 자연스러운 표현으로 개선해주세요

작문 주제: [주제 입력]
글의 종류: [일기/이메일/에세이/편지 등]
```

## 사용 팁

1. **매일 다른 프롬프트 사용**: 단조로움을 피하기 위해 매일 다른 유형의 프롬프트를 사용하세요
2. **구체적인 내용 입력**: [학습 내용] 부분에 오늘 배운 구체적인 단어나 문법을 입력하세요
3. **점진적 난이도 증가**: 처음엔 간단한 대화부터, 점차 복잡한 상황 대화로 발전시키세요
4. **정기적 복습**: 일주일에 한 번은 이전에 학습한 내용을 다시 연습하세요

---

💡 **팁**: 각 언어별 디렉토리(@languages/[언어명])에 학습 기록을 저장하고, 이 프롬프트들로 매일 10분씩 실전 연습을 해보세요!
